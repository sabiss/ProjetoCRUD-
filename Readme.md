# CRUD simples para clinicas médicas

### Projeto CRUD responsável por gerenciar pacientes de uma clínica médica

## Objetivo
Projeto tem como objetivo permitir o gerenciamento dos pacientes existentes no sistema. Tem como função cadastrar novos pacientes, atualizar cadastros, pesquisar paciente específico, listar todos os clientes ativos e inativos e desativar um cadastro

## Tecnologias
Foi usada apenas a linguagem de C++
